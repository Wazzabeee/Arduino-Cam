# Arduino-Cam

## Requirements

* Node.js v12.8.1
* Git

## Node.js setup

Clone the repository and install the dependencies.

```bash
git clone https://github.com/axel0070/Arduino-Cam.git
cd Arduino-Cam
npm install
```
Start the application
```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000).
